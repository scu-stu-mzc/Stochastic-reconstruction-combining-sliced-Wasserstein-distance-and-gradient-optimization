# Stochastic-reconstruction-combining-sliced-Wasserstein-distance-and-gradient-optimization
To perform 2D-to-2D microstructure reconstruction, run the Rec-2D-to-2D.py file.
To perform 2D-to-3D microstructure reconstruction, run the Rec-2D-to-3D.py file.
To perform 3D-to-3D microstructure reconstruction, run the Rec-3D-to-3D.py file.
Parameters such as the reference image path can be directly modified in the corresponding files.
